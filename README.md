# FrenchInputMethod
A cross platform french input method. (Actually just a plugin) 跨平台法语输入法

该输入方法参照汉语音标，

数字 `1` 代表分音符 `¨`
数字 `2` 代表闭音符 `´`
数字 `3` 代表长音符 `ˆ`
数字 `4` 代表开音符 `
数字 `5` 代表软音符 `¸`
数字 `6` 仅在 `o` 后使用，代表字母 `œ`

输入组合映射如下：
e + 1 --> ë
e + 2 --> é
e + 3 --> ê
e + 4 --> è
...
c + 5 --> ç
o + 6 --> œ
